This is a simple Note-taking REST API built with Spring Boot using Astrapay's project structure.

🚀 Features

1. Create, read, and delete notes

2. in-memory data

3. Clean project structure

🧑‍💻 How to Run

Make sure you have Java 17+ and Maven.

📦 API Endpoints Guide

🔍 Get All Notes

Request:
GET /notes

📝 Create a Note

Request:
POST /notes

❌ Delete a Note

Request:
DELETE /notes/{id}
